Game Interface

Start Menu

The start menu is a simple screen that allows the player to start the game by pressing Enter.

The game controls can be changed in the Unity launcher. By default the controls, for both keyboard & mouse and controller are:

  • Move - WASD or Left Analog
  • Jump - Space or X
  • Rotate camera - E/Q or R1/L1
  • Change form - R or Triangle
  • Attack - Left Mouse Click or Right Analog
  • Restart from last checkpoint - Enter or Start

Gameplay

During the gameplay the interface is supposed to be minimalistic to keep the focus of the player on the action.

The only element present in the user interface is an health bar that allows the player to keep track of the damage taken. The rest of the information is conveyed through the game's sound effects.

End Game

When the game ends a simple win screen is presented with the credits of the game.
